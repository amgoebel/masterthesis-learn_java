Kapitel 1 - Hallo Welt

Rechts siehst du ein "Hallo Welt" Programm. In vielen Programmiersprachen ist dies das erst Programm, das man schreibt. Es macht nichts weiter, als dass es die Worte "Hallo Welt" ausgibt.

Im ersten Kapitel lernst du mit Hilfe dieses einfachen Programms einige Regeln für Java kennen. Du wirst hier schon auf einige Begriffe stoßen, die am Anfang noch unklar sind und die du erst im Laufe dieses Lehrgangs richtig verstehen wirst. Mach dir darum keine Sorgen.

Das Programm besteht aus einer Klasse mit dem Namen "main":

public class main {
...
}

Da das Programm nur aus einer Klasse besteht, ist dies die Hauptklasse. Jede Java Datei hat die Erweiterung .java und besitzt eine Hauptklasse mit dem gleichen Namen wie die Datei. Dieses Programm würde also unter dem Namen "main.java" abgespeichert.

Du wirst noch genauer erfahren, was man bei der Programmierung unter einer Klasse versteht. Zunächst kannst du dir eine Klasse als ein Konzept mit einer Aufgabe vorstellen. Die Aufgabe dieser Klasse ist die Ausgabe der Texts "Hallo Welt". Klassennamen schreibt man in der Regel groß. Die geschweiften Klammern geben an, was alles zu der Klasse hinzugehört.

In der Klasse befindet sich eine Methode main(), die die Aufgabe des Programms enthält:

public static void main(String[] args) {
...
}

Der Name einer Methode endet immer mit runden Klammern. Später wirst du sehen, dass es auch Methoden gibt, bei denen nichts in der Klammer steht. Trotzdem müssen die runden Klammern immer am Ende des Namens vorhanden sein.
Wie bei der Klasse zeigen die geschweiften Klammern an, was zu der Methode hinzugehört.

Die Begriffe "public", "static", "void" und "String[] args" werden wir später genauer verstehen. Fürs erste reicht es zu verstehen, dass sie (notwendnige) Eigenschaften der Methode beschreiben.

Der entscheidende Befehl für das Programm ist:

System.out.println("Hallo Welt");

Dieser Befehl weist den Computer an, den Text "Hallo Welt" auszugeben. Der Befehl ist zusammengesetzt aus einzelnen Komponenten (zu erkennen an den Punkten). Du wirst bald verstehen, was die einzelnen Teile bedeuten. Fürs Erste kannst du dir merken, dass man mit diesem Befehl Text ausgeben kann.
