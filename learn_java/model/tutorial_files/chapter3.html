Kapitel ?? - Vererbung

Wir haben schon festgestellt, dass ein wichtiges Konzept der OOP die Wiederverwendbarkeit von Code ist.
Häufig kommt es vor, dass man zwei verschiedene Klassen benötigt, die sich nur in kleinen Teilen unterscheiden.

Damit sich Code nicht in zwei (oder mehr) Klassen wiederholt, erstellt man eine Basis-Klasse, die alle gemeinsamen Elemente enthält. Die benötigten Klassen sind dann Kopien dieser Basis-Klasse, erweitert um die jeweiligen (unterschiedlichen) Eigenschaften.

Die Basis-Klasse nennt man die Elter-Klasse und die abgeleiteten Klassen heißen Kind-Klassen. Die Eigenschaften der Elter-Klasse werden auf die Kind-Klassen vererbt.

Um anzuzeigen, dass eine Klasse von einer anderen Klasse erbt, verwendet man das Keywort extends.

Im folgenden Beispiel sind die Klassen Auto und Motorrad Kind-Klassen von der Klasse Fahrzeug:

class Fahrzeug {
  public String art = "Kraftfahrzeug";  // Attribut von Fahrzeug
  public void hupen() {                 // Methode von Fahrzeug
  System.out.println("Tuut, tuut!");
  }
}

class Auto extends Fahrzeug {
  public int anzahlRaeder = 4;     // Attribut von Auto
}

class Motorrad extends Fahrzeug {
  public int anzahlRaeder = 2;     // Attribut von Motorrad
}

class main{
  public static void main(String[] args) {
    // Deklaration von einem Auto und einem Motorrad
    Auto meinAuto = new Auto();
    Motorrad meinRad = new Motorrad();

    // Aufrufen der hupen() Methode (von der Fahrzeug Klasse)
    meinAuto.hupen();
    meinRad.hupen();

    // Anzeigen der Attribute Art und der Anzahl Räder
    System.out.print("Art: " + meinAuto.art);
    System.out.println(", Anzahl Räder: " + meinAuto.anzahlRaeder);
    System.out.print("Art: " + meinRad.art);
    System.out.println(", Anzahl Räder: " + meinRad.anzahlRaeder);
  }
}

Die Ausführung dieses Programms ergibt folgenden Output:

Tuut, tuut!
Tuut, tuut!
Art: Kraftfahrzeug, Anzahl Räder: 4
Art: Kraftfahrzeug, Anzahl Räder: 2

Die Klassen Auto und Motorrad besitzen beide auch das Attribut art und die Methode hupen().
