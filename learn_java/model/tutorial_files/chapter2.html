Kapitel ?? - Eingabe

Häufig ist es notwendig, dass ein Programm Informationen vom Nutzer während der Laufzeit erhält.

In Java gibt es dafür eine sogenannte Scanner Klasse, die diese Funktionalität bereit stellt. Diese muss zu Beginn des Programms importiert werden mit: 

import java.util.Scanner;

Im Programm kann man dann (wie bekannt) ein Objekt der Klasse Scanner erstellen mit:

Scanner scanner = new Scanner(System.in);

System.in gibt dabei an, dass der Scanner den standard input stream (meistens die Kommandozeile) als Quelle verwenden soll. Möglich wären hier auch einfach eine String Variable oder eine Datei.

Jetzt kann man den scanner dazu verwenden, um Text vom Nutzer während der Laufzeit zu erhalten. Mit dem Befehl

String text = scanner.nextLine();

wird eine Zeile von der Quelle (im diesen Fall die Eingabe des Nutzers) eingelesen und als Wert der Variable text zugeordnet.

Hinweis: Die Eingabe des Nutzers wird durch die Enter Taste auf der Tastatur beendet.
